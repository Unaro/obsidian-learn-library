# Что такое MVT в Django?

## Вопрос
Что такое архитектурный паттерн MVT в Django? Чем отличается от MVC?
?
## Ответ

**MVT (Model-View-Template)** — вариация MVC, используемая в Django:

**Model (Модель):**
- Структура базы данных
- Бизнес-логика и работа с данными
- ORM для запросов к БД

**View (Представление):**
- Обработка HTTP-запросов
- Извлечение данных через Модель
- Возврат HTTP-ответа или передача данных в Шаблон
- **Аналог Controller в MVC**

**Template (Шаблон):**
- HTML-разметка
- Отображение данных пользователю
- **Аналог View в MVC**

**Отличие от MVC:**
```
MVC: Model → View ← Controller
MVT: Model → Template ← View

В Django View принимает решение, какие данные показать,
а Template только отображает (пассивный)
```

**Пример:**
```python
# View
def user_list(request):
    users = User.objects.all()
    return render(request, 'users.html', {'users': users})

# Template (users.html)
{% for user in users %}
    <p>{{ user.username }}</p>
{% endfor %}
```
