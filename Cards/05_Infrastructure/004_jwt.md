# Что такое JWT?

## Вопрос
Что такое JWT (JSON Web Token)? Из чего состоит?
?
## Ответ

**JWT (JSON Web Token)** — компактный формат передачи данных об авторизации.

**Структура:** `Header.Payload.Signature`

**1. Header:**
```json
{
  "alg": "HS256",
  "typ": "JWT"
}
```

**2. Payload:**
```json
{
  "sub": "1234567890",
  "username": "john",
  "role": "admin",
  "iat": 1516239022,
  "exp": 1516242622
}
```

**3. Signature:**
```
HMACSHA256(
  base64UrlEncode(header) + "." + base64UrlEncode(payload),
  secret
)
```

**Пример токена:**
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.
eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.
SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
```

**Преимущества:**
- Stateless (не нужно хранить на сервере)
- Компактный размер
- Подпись защищает от подделки

**Недостатки:**
- Нельзя отозвать (нужен blacklist)
- Размер больше чем у сессионных токенов
