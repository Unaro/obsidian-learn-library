# SDLC этапы разработки

## Вопрос
Какие этапы SDLC (Software Development Life Cycle)?
?
## Ответ

**SDLC** — жизненный цикл разработки ПО:

**1. Сбор и анализ требований:**
- Перевод бизнес-требований в спецификации
- BPMN-схемы, Use Cases
- **Результат:** документ требований

**2. Проектирование (Design):**
- Выбор стека технологий
- UML-диаграммы, схема БД
- Архитектура API
- **Результат:** технический проект

**3. Разработка (Implementation):**
- Написание кода
- Code review
- **Результат:** рабочий продукт

**4. Тестирование (Testing):**
- Unit, Integration, E2E тесты
- **Результат:** отчёт о тестировании

**5. Развертывание (Deployment):**
- Docker, CI/CD
- **Результат:** продукт на проде

**6. Поддержка (Maintenance):**
- Мониторинг, фикс багов
- Рефакторинг
- **Результат:** обновления продукта

**Модели:**
- Waterfall (каскадная)
- Agile (итеративная)
- DevOps (непрерывная)
